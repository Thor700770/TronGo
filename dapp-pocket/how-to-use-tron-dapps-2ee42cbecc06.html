How to use TRON Dapps?

葉顧純
Jun 7, 2019 · 5 min read

TRON is a blockchain platform launched as the foundation for a decentralized entertainment ecosystem. TRON aims to expand the market of decentralized digital content applications by making it easier to create and deploy them.

The CEO of it, Justin Sun, announced on Apr 13, 2019 that there had been 2.5 million active users on TRON Mainnet, which indicated the success of TRON’s network pluralizing. Three characteristics of TRON — Speed, Low Transaction Fees, and Good Scalability, can be the reasons lead to its noticeable progress. So there are many Dapp (Decentralized App) based on TRON.

How to use TRON Dapps?

Here, you might have grown some interest about the tech-revolutionary OS, TRON. The next step is to start your journey exploring the world of Dapps, but, how to?

We all have experiences of using apps on our phones. If we want to use apps from App store, we have to first, have the app “App Store”, and second, create an apple account so that we are legitimate users for using these apps. Similar to having an apple account for ios apps, people need a “wallet” for using cryptocurrency TRON Dapps.

There are 2 ways to get the “wallet”, namely, installing the wallet plugin on a browser, and downloading the wallet app on mobile.

TronLink — the most popular TRON web wallet

Tronlink is the first and most popular TRON wallet, which was launched at TRON’s official website and supported by the TRON foundation. TronLink functions as a TRX wallet similar to MetaMask for Ethereum in the form of a browser extension. The browsers that TronLink works with are Chrome and Brave.

Step 1. Install TronLink on your browser

As long as you have had one of the browsers, you can find TronLink in Web Store ( Here we take Chrome Website as an example. )

After you click on “ add to Chrome”, a confirm box will pop up asking if you want to “add TronLink this extension”.

Once you install it successfully, you can see the TronLink on your browser.

Step 2. Create a TRON account

Type a name for your “Account name”. You can probably type anything you want without worrying others have already taken the name. However, you can not choose the same name for different accounts in the same TronLink. ( The Account name is only for you to distinguish multiple accounts on your computer. )

Next pop-up box asks you to Create Account or Restore Account. For Create Account, you will get a set of mnemonic phrase, which is a kind of word password for logging in the account ever since. Please make sure you write down or keep your recovery phrase safely in any way you prefer. Once you lose it, you might need to say goodbye to all your assets forever too!!

Step 3. Start to use Dapps!

🎉Congratulations!

You can start transferring cryptocurrency on TRON Dapps through websites. Since TronLink allows users to import several accounts, you can switch to another account easily just by clicking on it.

Dapp Pocket — the first TRON/Ethereum mobile wallet that created for Dapps lovers

Aside from using TRON plugin wallet, we can also use a TRON mobile wallet, Dapp Pocket.

With Dapp Pocket, you can manage cryptocurrency easily and securely on phones. Moreover, you can use every TRON & Ethereum Dapps in Dapp Pocket. Also, it can help you find the best price to exchange cryptocurrency.

Step 1. Download Dapp Pocket

You can get Dapp Pocket on the App Store (iOS) or Google Play Store (Android).

Step 2. Create a TRON account

Open the app. You can choose to Create a New Account or Import your Account.

Just like creating an account on TronLink, the system will pop up a set of recovery phrase once you click on creating a new account. You can back up your account phrase right away by the button below the phrase.

Next, enter a PIN code for confirming your transaction in the future.

With Dapp Pocket, you can also get both TRON and Ethereum account at one time!
